01

项目背景：传统品牌咨询为什么难以规模化？

品牌创意高度依赖资深顾问经验，策略产出不稳定、评审主观化、资产一次性、能力断层化。

传统方式痛点

  • 经验黑箱：策略质量高度依赖资深顾问个人经验，难以编码传承。
  • 产出不稳：同一品牌，不同顾问策略差异巨大，缺乏标准流程。
  • 评审主观：提案改稿多轮，标准随评审人变化，缺乏客观依据。
  • 资产一次：项目结束留下PPT，没有形成可复用的方法和模板。
  • 能力断层：新人成长慢，老人被琐事困住，经验传递困难。
  • 价值后置：顾问常在需求明确后介入，影响力集中在执行而非策略定义。

BrandCreative Agent 解决方案

  • 方法论驱动：将品牌屋、竞品矩阵、用户洞察转成稳定的策略框架。
  • 8Agent协同：需求理解→竞品分析→需求升维→策略生成→创意概念→设计语言→方案输出→评估交付。
  • Checkpoint确认：5个关键节点人类确认，确保方向不偏离。
  • 人类校准：AI 负责结构化，顾问负责价值判断和质量放行。
  • 资产沉淀：输出策略文档、竞品报告、创意概念、设计规范、知识库。
  • 组织赋能：把高级顾问经验转化为团队可复制的品牌创意基础设施。

03

人机协同：AI 做结构化，顾问做判断

AI 提升创意效率，但策略价值、方向选择和质量责任仍由顾问掌控。

AI Agent 负责

读取资料、整理定位、生成分析、拆解策略、创意概念、设计规范、多版本方案、评估报告。

高级顾问负责

定义品牌目标、校准用户画像、判断竞品洞察、选择创意方向、把控设计语言、确认最终质量。

协同工作方式

每个Agent产物经确认后进入下一步，发现策略偏离或创意不足时及时纠偏，确保质量可控。

协同工作三大原则 让 AI 的结构化能力与顾问的专业判断形成互补。

Checkpoint确认

C1→C2→C4→C3→C5 五个关键节点必须人类确认，避免错误判断在后续环节累积。

人工校准

当策略偏离业务目标、分析过于表面、创意缺乏差异化时，由顾问纠偏回到品牌价值。

方法论闭环

每个策略建议必须有品牌屋依据、竞品支撑、用户洞察，避免只有观点没有证据。

06

专业知识库：把品牌创意经验转成 Agent 资产

高级顾问的隐性经验被拆解为策略库、分析框架、创意模板、设计规范、评估标准和知识库。

品牌策略库

品牌定位画布、北极星指标、用户心智模型、竞品分析框架、需求升维方法。

分析框架库

竞品定位矩阵、视觉风格分析、策略差异识别、机会点挖掘、用户价值主张。

创意概念模板

创意方向定义、符号系统设计、情绪板制作、视觉语言提炼、差异化表达。

设计规范模板

设计语言定义、色彩系统、字体系统、符号规范、应用规范、生图提示词。

评估标准库

策略一致性、创意差异化、品牌匹配度、落地完整度、可执行性评估。

知识库规范

Markdown便于评审阅读，Excel便于分派跟踪，HTML便于分享传播。
