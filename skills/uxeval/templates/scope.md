## 校验

填完后跑：

```bash
designos input check uxeval
```

会校验：
- [x] 评估目标 ≥ 1 句
- [x] 重点角色 ≥ 1 个
- [x] 重点模块 ≥ 1 个
- [x] 不评估的部分 ≥ 1 项（强制划界）
- [x] 不含敏感信息

通过后即可：

```bash
designos run uxeval --mode web    # 或 --mode client
```
