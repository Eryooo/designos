# [SCAFFOLD] 数据看板 PRD

> ⚠️ 本文件为 scaffold 占位。P1 补充真实内容。

## 1. 产品概述

为业务负责人提供实时业务数据看板，支持多维度 KPI 监控和数据下钻分析。

## 2. 用户角色

- **业务负责人**: 看总览，做决策
- **数据分析师**: 深度下钻分析
- **运营人员**: 看自己负责的指标

## 3. 核心功能

- 多维度 KPI 卡片（营收/转化/留存）
- 时间维度筛选（日/周/月/自定义）
- 数据下钻（总览→明细）
- 数据导出
- 异常告警

## 4. 非功能需求

- 首屏 KPI 加载 <2 秒
- 关键 KPI ≤2 click 可达
- 支持移动端响应式
