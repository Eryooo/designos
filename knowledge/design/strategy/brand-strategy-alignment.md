# 品牌策略对齐法（Brand Strategy Alignment · M01）

> 通用资产 · `design.strategy.brand-strategy-alignment` · status: pilot
> IP 设计第一阶段。把产品/用户/竞品/价值收敛成一份可向下传导的品牌 brief。

## purpose

为整条 IP 设计链确立上位约束:北极星、用户画像、核心价值、品牌人格关键词、竞品差异化。它决定后续世界观、人格、视觉是否有方向。

## input_contract

- 产品/服务定义与业务目标。
- 目标用户描述(职业、场景、痛点),有真实研究优先。
- 竞品 IP 信息(≥3 个)。
- 品牌已有调性/词库(可选)。

## decision_framework

1. **品牌北极星**:一句话,必须包含用户对象、方式、能力、价值、结果五要素,且落在情感/体验价值而非功能。
2. **用户画像**:年龄/职业/典型一天/核心痛点/期待感受;基于真实研究,推断处标注。
3. **核心价值四层**:产品价值 / 用户价值 / 组织价值 / 品牌价值,每层 3–5 项。
4. **品牌人格关键词**:3–5 个,每个给"为什么选"的依据(关联北极星或用户痛点)。
5. **竞品差异化矩阵**:竞品 IP 形象 × 人格 × 视觉风格 × 差异化机会,客观不贬低。

## expert_heuristics

- 北极星是后续一切的锚,模糊的北极星会让每个下游阶段反复返工。
- 人格关键词控制在 3–5 个;超过 5 个等于没有重点。
- 核心价值要能反推到视觉与内容,否则只是 PPT 话术。
- 差异化机会来自竞品空白,不是自夸。

## output_contract

- 一份 brand brief(见 `design.templates.brand-brief`),含上述五部分。
- 人格关键词清单(下游 M03/M04 直接消费)。

## quality_rubric

| 分级 | 信号 |
|---|---|
| 高阶可评审 | 北极星五要素齐全且为情感价值,人格词有依据,差异化基于竞品空白 |
| 中阶可用 | 五部分齐全,但北极星偏功能或人格词依据弱 |
| 低阶/不合格 | 北极星是功能描述,用户画像臆造,竞品分析靠贬低 |

## common_failure_modes

1. 北极星写成功能("帮你写周报"),不是体验价值。
2. 用户画像凭空臆造,无痛点或痛点不真实。
3. 人格关键词超过 5 个或无选择依据。
4. 竞品分析贬低对手,得不到真实差异化。
5. 核心价值与后续视觉/内容无法对应,沦为口号。

## senior_review_checklist

- [ ] 北极星含五要素且为情感/体验价值。
- [ ] 用户画像有具体痛点,推断已标注。
- [ ] 人格关键词 3–5 个且每个有依据。
- [ ] 竞品 ≥3 且差异化来自空白而非贬低。
- [ ] 核心价值可反推到视觉/内容。

## examples_to_read

- 案例引用为可选项，且不得指向任何私有业务证据或项目专属 case 路径。

## do_not_claim

- 不替代真实用户研究与市场调研。
- 不替代商业战略与法务合规判断。
- 不保证差异化定位的市场成功。
