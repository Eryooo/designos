# 品牌物料落地法（Brand Material Realization · M06）

> 通用资产 · `design.ip.brand-material-realization` · status: pilot
> IP 设计第六阶段。把视觉规范落成可生产、可控成本、不崩一致性的物料体系。

## purpose

让 IP 从屏幕走向应用与实体:品牌应用规范、衍生品、宣传物料、传播指南、生产清单。它把"视觉好看"约束成"可落地且一致",并显式管理落地成本与一致性风险。

## input_contract

- 视觉规范 + 核心符号 + 风格坐标 + 禁忌项(M04)。
- 人格与声音边界(M03,用于传播语调)。
- 物料偏好与预算/产能约束(可选)。

## decision_framework

1. **品牌应用规范**:IP 与 logo 联合使用(组合/位置/最小尺寸/安全区)、形象使用规范(各场景推荐形态/尺寸/色彩模式)、禁止使用示例(≥5,继承 M04 禁忌)。
2. **衍生品清单**:品类 × 名称 × 设计要点 × 材质 × 工艺 × 尺寸;每项评估**可落地性与成本**,避免过度复杂。
3. **宣传物料规范**:社媒/活动物料的尺寸、版式层级、留白原则;标注 CMYK/RGB/Pantone 色彩模式。
4. **传播指南**:场景化色彩情绪 + 语调(与人格声音边界一致)+ IP 在不同场景的角色(主视觉/向导/伙伴/符号)。
5. **生产清单**:首批推荐物料(名称/数量/周期/预算)+ 生产建议(打样→材质确认→批次→版权登记提示)。

## expert_heuristics

- 一致性在物料阶段最容易崩:靠"核心符号 + 风格坐标 + 色彩模式"三锁。
- 衍生品要在"识别度"和"成本"间取舍,复杂工艺未必值得。
- 印刷物料必须给色彩模式(CMYK/Pantone),只给屏幕 RGB 会偏色。
- 传播语调要回到人格,不同场景换语气但不换人设。

## output_contract

- 物料规格手册(见 `design.templates.brand-material-spec`):应用规范 + 衍生品清单 + 宣传物料 + 传播指南 + 生产清单。

## quality_rubric

| 维度 | 高阶可评审 | 中阶可用 | 低阶/不合格 |
|---|---|---|---|
| 应用规范 | 联合使用 + 禁止示例齐全 | 有规范但缺安全区/禁止 | 无规范 |
| 衍生品 | 含可落地性与成本评估 | 有清单无成本考量 | 清单过度复杂不可产 |
| 色彩模式 | 标注 CMYK/RGB/Pantone | 仅 RGB | 无色彩模式 |
| 传播语调 | 与人格一致且分场景 | 有语调但不分场景 | 与人格脱节 |
| 一致性 | 三锁明确 | 部分锁定 | 物料风格各异 |

## common_failure_modes

1. 物料各做各的,无统一应用规范,风格崩。
2. 衍生品过度复杂,工艺成本不可落地。
3. 印刷物料只给 RGB,实际生产严重偏色。
4. 传播语调与人格脱节,各场景人设不一。
5. 无禁止使用示例,IP 被错误拉伸/遮挡/变色。

## senior_review_checklist

- [ ] IP+logo 联合使用规范含最小尺寸与安全区。
- [ ] 禁止使用示例 ≥5,继承 M04 禁忌。
- [ ] 衍生品清单含可落地性与成本评估。
- [ ] 物料标注 CMYK/RGB/Pantone。
- [ ] 传播语调与人格声音边界一致且分场景。
