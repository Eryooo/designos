# IP 设计方法论总纲（IP Design Methodology）

> 通用资产 · `design.ip.methodology` · status: pilot
> 这是 IP 设计的六阶段总纲,定义阶段划分、依赖关系与跨阶段判断逻辑。
> 目标:让 AI 按本方法论产出的 IP 设计,至少可替代中低阶设计师,
> 并达到高阶设计师"愿意在其上评审/微调"而非推倒重来的质量。

## purpose

把"IP 设计"从模糊创意活动,拆成可决策、可评审、可反推质量的六个阶段,并明确每个阶段不是执行步骤而是设计判断。它统一上层流程,各阶段深法在对应方法论文件。

## input_contract

- 产品/服务定义、目标用户、核心价值、竞品信息(进入 M01 所需)。
- 用户对文化原型、性格倾向、视觉气质、内容渠道、物料偏好的偏好(可选,缺失则推断并标注)。
- 缺失关键输入时不臆造,标 `[inferred]` 或在对应阶段 gap 中记录。

## decision_framework

六阶段与依赖:

| 阶段 | id | 名称 | 依赖 | 核心产出 |
|---|---|---|---|---|
| 1 | M01 | 品牌策略对齐 | — | 品牌 brief |
| 2 | M02 | 世界观构建 | M01 | 世界观设定 |
| 3 | M03 | 人格塑造 | M01 | 人格档案 + 声音/行为边界 |
| 4 | M04 | 视觉转化 | M01/M02/M03 | 视觉规范 + 绘图提示词 |
| 5 | M05 | 内容叙事 | M01/M03 | 内容规划 |
| 6 | M06 | 品牌物料落地 | M04/M03 | 物料规格手册 |

跨阶段判断主线:
1. **策略生长而非凭空创造**:IP 从用户需求 + 文化母体中长出来,北极星必须是情感/体验价值,不是功能描述。
2. **一致性向下传导**:每阶段关键词(人格词、世界观词)必须能在下游视觉/内容/物料里找到对应落点,断链即返工。
3. **可评审优先于齐全**:每阶段产出要能被资深设计师按 rubric 打分,而不是"写满了"。

## expert_heuristics

- 阶段之间是契约关系:上游关键词是下游的输入约束,下游不能自由发挥。
- 任一阶段达不到中阶质量,宁可写入 `design.quality.professional-gap-report`,不假装完成。
- "酷"不是目标,服务人格与识别度才是。
- 文化原型(武侠/西游/赛博等)是通用语言,具体品牌符号是项目专属,不能互相混淆。

## output_contract

- 六个阶段各自的结构化产出(见各阶段方法论与 `design.templates.*`)。
- 一份跨阶段一致性说明:关键词从 M01 到 M06 的传导链。
- 一份质量自评(对照 `design.quality.ip-design-quality-rubric`)。

## quality_rubric

| 分级 | 信号 |
|---|---|
| 高阶可评审 | 六阶段关键词传导无断链,每阶段 rubric ≥ 良,gap 报告诚实 |
| 中阶可用 | 阶段齐全、主线清楚,但部分阶段关键词传导弱或证据不足 |
| 低阶/不合格 | 阶段拼凑、关键词各说各话、功能当价值、无质量自评 |

## common_failure_modes

1. 北极星写成功能描述("帮用户做 X"),不是情感/体验价值。
2. 各阶段关键词互不相关,人格词在视觉里找不到落点。
3. 跳过 M01/M02 直接做视觉,IP 没有内在动机与世界。
4. 把项目专属符号当成通用方法,污染可复用层。
5. 阶段达不到中阶质量却标记完成,不写 gap。
6. 只产出流程清单,没有任何设计判断与取舍理由。

## senior_review_checklist

- [ ] 北极星是情感/体验价值且含用户对象、方式、能力、价值、结果。
- [ ] 人格词、世界观词在视觉/内容/物料中都有可指认的落点。
- [ ] 每阶段都有质量自评且与 rubric 对应。
- [ ] 不足阶段已写入 professional-gap-report。
- [ ] 通用产出无项目专属品牌词。

## examples_to_read

- 案例引用为可选项，且不得指向任何私有业务证据或项目专属 case 路径。
- 如需案例支撑，使用已批准的合成设计案例（synthetic case；后续在 `eval/golden-cases/` 下补充）。

## do_not_claim

- 不声明可替代法务、商标/版权审查。
- 不声明产出即最终商用视觉资产。
- 不声明替代真实用户研究;用户画像在无研究数据时为推断。
- 不保证生成图像的版权清洁性。
