# 视觉转化法（Visual Translation · M04）

> 通用资产 · `design.visual.visual-translation` · status: pilot
> IP 设计第四阶段,也是质量最吃重的一阶。把人格/世界观翻译成可识别、可延展、
> 可落地的视觉系统。本文是**视觉决策系统**,不是"人格词→形色质动"的查表。

## purpose

让视觉不是好看,而是**正确**:能承载人格、能在各尺寸被识别、能继承品牌基因、能延展出状态与物料、能被 AI 稳定复现。它解决旧方法论的核心短板——只有通用映射,没有冲突取舍、识别度标准、品牌基因继承。

## input_contract

- 人格行为模式与关键词(M03)、世界观关键词(M02)、北极星与差异化(M01)。
- 既有品牌资产(若是已有品牌的 IP):主色、logo、字体、已有视觉符号——用于基因继承。
- 色彩/风格偏好与使用场景清单(尺寸、端、明暗)(可选)。

## decision_framework

### 1. 品牌基因继承(已有品牌时必做)
- 先盘点既有品牌的"不可动基因":主色、核心符号、字体气质、造型语言。
- IP 视觉必须在基因之上生长:继承可识别基因 + 注入 IP 专属符号,而非另起炉灶。
- 明确"继承什么 / 新增什么 / 绝不冲突什么"三栏。
- 全新品牌则跳过继承,但仍需定义"未来不可动基因",供后续延展锁定。

### 2. 形态设计(人格→形,带冲突取舍)
- 人格关键词映射几何倾向(稳定→圆;力量→三角;柔和→有机)。
- **冲突取舍**:多个人格词形态冲突时,定主形 + 辅形,不并列堆砌;给"为什么主形是它"的依据(通常对应北极星最强的那个词)。
- 锁定 1 个核心识别符号(主符号承担约 60–70% 识别权重)+ 1–2 个辅助符号。

### 3. 色彩体系(60-30-10 + 可访问性)
- 主色 60% / 辅色 30% / 点缀 10%,每色给十六进制 + 使用场景。
- 对比度满足可读(正文级元素对照 WCAG AA),深浅场景都验证。
- 色彩与世界观/人格情绪挂钩,不是审美偏好。

### 4. 材质 / 光影
- 材质映射世界观(云原生→毛玻璃;金属/丝绸→力量等)。
- 固定主光源方向,定义光影层次,保证跨图一致。

### 5. 识别度与小尺寸可用性(旧库缺失,必补)
- **识别度量化**:核心符号在 32px 下必须清晰可辨;远观能凭轮廓认出。
- **尺寸分级简化**(给出每级画法):
  - 大(≥128px):完整形态与细节。
  - 中(48–128px):标准形态,弱化次要细节。
  - 小(24–48px):简化为核心符号 + 主色。
  - 极小(<24px):退化为单符号/单色标记。
- 每一级都要可识别为同一 IP,不能换形象。

### 6. 状态延展(旧产物的强结构,反抽并体系化)
- 定义关键状态序列(如待机/激活/执行/确认/完成/失败),每状态给:视觉表现 + 时长 + 应用场景。
- 状态要与 `design.persona.voice-and-behavior-boundary` 的情绪表达一一对应。

### 7. 风格谱系定位(旧库缺失,必补)
- 在风格谱系上明确坐标:扁平 ↔ 拟物、几何 ↔ 有机、极简 ↔ 繁复、成熟商务 ↔ 萌系、2D ↔ 3D。
- 谱系坐标决定提示词与延展边界,避免不同物料风格漂移。

### 8. 严格禁忌项(旧产物的强结构,反抽)
- 明确"绝不出现"清单:破坏基因色、遮挡核心符号、负面表情、拉伸变形、风格越界(如不要 3D/不要二次元)。
- 禁忌项直接进 M06 物料规范与提示词避免项。

## expert_heuristics

- 识别度 > 精美:一个在 32px 糊掉的精致形象是失败的。
- 形态服务人格,不为酷而设;删掉某元素若不影响识别,就该删。
- 已有品牌做 IP,基因继承是第一约束,不是可选项。
- 状态/尺寸/物料的一致性,靠"锁定核心符号 + 风格坐标",不靠每次重画。
- 色彩点缀 10% 决定"科技/能量感",但不能喧宾夺主。

## output_contract

- 视觉规范草案(见 `design.templates.visual-spec`):基因继承 + 形态 + 色彩 + 材质 + 识别度/尺寸分级 + 状态延展 + 风格坐标 + 禁忌。
- 核心符号定义 + 风格坐标 + 禁忌清单(下游 `design.visual.image-prompt-system`、M06 消费)。

## quality_rubric

| 维度 | 高阶可评审 | 中阶可用 | 低阶/不合格 |
|---|---|---|---|
| 品牌基因继承 | 继承/新增/禁冲突三栏清晰 | 提到基因但不系统 | 无视既有品牌另起炉灶 |
| 形态-人格映射 | 主辅形有取舍依据 | 有映射但堆砌 | 形态与人格无关 |
| 识别度 | 32px 可辨且远观可认 | 标准尺寸可识别 | 小尺寸糊成一团 |
| 小尺寸分级 | 四级简化画法齐全 | 有简化但不分级 | 无小尺寸方案 |
| 状态延展 | 状态与情绪一一对应 | 有状态但缺时长/场景 | 无状态延展 |
| 风格谱系 | 坐标明确锁定延展 | 风格大致清楚 | 风格漂移 |
| 禁忌项 | ≥5 条可判定 | 有禁忌但笼统 | 无禁忌 |

## common_failure_modes

1. 只做"人格词→形色质动"通用映射,无冲突取舍,主形不明。
2. 缺识别度标准,小尺寸下形象糊掉/不可辨。
3. 无小尺寸分级简化,各尺寸各画一套,识别不统一。
4. 已有品牌却不继承基因,IP 与品牌割裂。
5. 无风格谱系坐标,物料之间风格漂移。
6. 无严格禁忌项,AI 生成时跑偏(3D/二次元/动物化)。
7. 色彩靠审美偏好,不挂钩人格/世界观,且不验证对比度。

## senior_review_checklist

- [ ] 已有品牌:基因继承三栏(继承/新增/禁冲突)齐全。
- [ ] 主符号承担主要识别权重,32px 可辨。
- [ ] 小尺寸四级简化画法齐全且同一形象。
- [ ] 色彩 60-30-10 且对比度达标。
- [ ] 状态延展与声音情绪一一对应。
- [ ] 风格谱系坐标明确。
- [ ] 严格禁忌 ≥5 条且可判定。
