# 28. 视觉锁定 (Visual Lock Definition)

> **方法论级别**: Senior Designer Standard  
> **核心命题**: 关键特征锁定 = 可识别性

## 定义与目的

视觉锁定是定义IP的关键视觉特征（design locks），确保在任何变体中都能识别。

### 资深 vs 初级

**初级**：每次画都不一样  
**资深**：锁定核心特征（轮廓/色彩/标志性元素），变体中保持

核心要素：轮廓锁、色彩锁、符号锁、比例锁、表情系统
