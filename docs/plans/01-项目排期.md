## 6. 风险与缓冲

| 风险 | 影响 | 缓冲策略 |
|---|---|---|
| Prompt 调优耗时超预期 | M1 延期 | W5 有 1 周 buffer；可先用 80 分 Prompt 发 beta |
| MCP Server 某个工具难实现 | M1 延期 | browser-capture 可降级为手动截图 |
| 种子用户反馈大改 | M2 延期 | W12 有 1 周 buffer |
| 法务审批慢 | M3 延期 | W13 就启动法务流程，不等代码完成 |
